# 📊 FinSmart Pro: Financial Insight Hub

**FinSmart Pro** is a comprehensive web-based financial toolkit designed to simplify budgeting, expense management, and professional taxation calculations. Built specifically with commerce students and finance enthusiasts in mind.

🚀 **Live Demo:** [https://sweetysingh09.vercel.app](https://sweetysingh09.vercel.app)

---

## ✨ Key Features

- **50/30/20 Budgeting Rule:** Automatically splits income into Needs, Wants, and Savings.
- **CMA Academy Corner:** A dedicated section for commerce students featuring a GST Calculator and professional study tips.
- **Interactive Visualizations:** Real-time Pie/Doughnut charts to visualize spending power using Chart.js.
- **Smart Report Export:** Download your financial analysis as a professional PDF report.
- **Responsive Dashboard:** Sidebar navigation for a seamless user experience across devices.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
- **Logic:** Vanilla JavaScript (ES6+)
- **Libraries:** - [Chart.js](https://www.chartjs.org/) for data visualization.
  - [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/) for PDF generation.
- **Deployment:** Vercel

## 📖 How to Use

1. Enter your Monthly Income in the Dashboard.
2. View the real-time breakdown of your finances in the grid and chart.
3. Switch to the **CMA Corner** for GST calculations.
4. Click **Download Report** to save your analysis for future reference.
